# factura-movil-android

[![Software License][ico-license]](LICENSE.md)

Esta aplicación proporciona una interfaz intuitiva y amigable para usuarios que deseen gestionar sus facturas y transacciones desde sus dispositivos Android.

## Características Principales:

- **Gestión de Clientes**: Registra y administra fácilmente la información de clientes, incluyendo datos de contacto y detalles comerciales.

- **Creación de Facturas**: Permite la creación rápida y personalizada de facturas con detalles específicos de productos o servicios, fechas de vencimiento y términos de pago.

- **Historial de Transacciones**: Mantiene un registro detallado de todas las transacciones realizadas, ofreciendo una visión completa del historial financiero.

- **Generación de Informes**: Ofrece herramientas para generar y personalizar informes financieros detallados y exportar en diferentes formatos.

- **Múltiples Monedas**: Admite transacciones en diferentes monedas, incluyendo Bitcoin.

- **Múltiples Empresas**: El usuario puede crear tantas empresas como necesite y realizar la gestión de cada una de ellas de manera independiente.

## Tecnologías Utilizadas:

- Desarrollado en lenguaje Kotlin para Android.
- Utiliza SQLite para la gestión eficiente de la base de datos local.
- Interfaz de usuario atractiva y fácil de usar, implementada con Material Design.

## Instrucciones de Uso:

1) Clona el repositorio en tu entorno de desarrollo Android Studio.
1) Cambia el `run configuration` to `app`.
1) Ejecuta la aplicación en tu dispositivo Android o emulador.
